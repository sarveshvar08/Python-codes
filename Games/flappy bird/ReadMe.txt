Flappy Bird – Python (Pygame)

This is a simple Flappy Bird game created using Python and Pygame.
The game includes bird animation, moving pipes, scoring, collision detection,
and the ability to restart after a game over.

Features:

Classic Flappy Bird gameplay

Tap SPACE to make the bird jump

Random pipe generation

Collision detection with pipes and ground

Animated bird (3-frame flapping)

Score and high score display

Smooth background and floor scrolling

Restart option after game over

How to Run:

Install Python

Install Pygame:
pip install pygame

Run the game:
python flappy.py

Controls:

SPACE → Jump

SPACE (after game over) → Restart

Assets Used:

Background

Floor

Bird sprites (up, mid, down)

Pipe sprite

Game over image
